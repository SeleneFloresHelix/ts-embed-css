Integration Code

To use these styles in your ThoughtSpot embedded application:

import { init, AuthType } from '@thoughtspot/visual-embed-sdk';

init({
  thoughtSpotHost: "https://your-instance.thoughtspot.cloud",
  authType: AuthType.TrustedAuthToken,
  customCssUrl: "https://your-domain.com/css/liveboard-starter.css"
});

Or use one of the theme variations:

  • css/themes/dark-theme.css - Modern dark theme
  • css/themes/light-minimal.css - Clean minimal theme
  • css/themes/corporate-blue.css - Professional corporate theme

Try Different Themes

To see how different themes look, replace the CSS file link in the <head> section:

<!-- Starter Theme (default) -->
<link rel="stylesheet" href="../css/liveboard-starter.css">

<!-- Dark Theme -->
<link rel="stylesheet" href="../css/themes/dark-theme.css">

<!-- Light Minimal Theme -->
<link rel="stylesheet" href="../css/themes/light-minimal.css">

<!-- Corporate Blue Theme -->
<link rel="stylesheet" href="../css/themes/corporate-blue.css">
