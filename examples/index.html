Custom Typography

All text on this page uses the TT Norms Pro Variable font, a modern variable font that supports a wide range of weights (100-900). The font provides a professional, clean appearance suitable for data visualization and analytics dashboards.

Font Configuration: 
@font-face {
  font-family: 'TT Norms Pro Variable';
  src: url('https://helix-builder.staging.helix.com/assets/TTNormsProVariable-b8ab4b19.ttf') format('truetype');
  font-weight: 100 900;
  font-display: swap;
}

:root {
  --ts-var-root-font-family: 'TT Norms Pro Variable', -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif;
}

Chart Color Palette

These are the 5 customizable chart colors used in visualizations:

Customization Example: 
:root {
  --ts-var-viz-color-1: #1976d2;  /* Primary blue */
  --ts-var-viz-color-2: #42a5f5;  /* Light blue */
  --ts-var-viz-color-3: #64b5f6;  /* Lighter blue */
  --ts-var-viz-color-4: #90caf9;  /* Pale blue */
  --ts-var-viz-color-5: #bbdefb;  /* Very pale blue */
}

Table Color Customization:

The following CSS variables control table appearance:

:root {
  /* Table header styling */
  --ts-var-table-header-background: #f5f5f5;
  --ts-var-table-header-color: #333333;
  --ts-var-table-header-border: #e0e0e0;
  
  /* Table row styling */
  --ts-var-table-row-background: #ffffff;
  --ts-var-table-row-alt-background: #fafafa;  /* Alternating rows */
  --ts-var-table-row-hover-background: #f5f5f5;  /* Hover state */
  
  /* Table text and borders */
  --ts-var-table-border-color: #f0f0f0;
  --ts-var-table-text-color: #333333;
}

Tips: Try hovering over the table rows to see the hover effect. Notice the alternating row backgrounds for better readability.

Integration Code

To use these styles in your ThoughtSpot embedded application:

import { init, AuthType } from '@thoughtspot/visual-embed-sdk';

init({
  thoughtSpotHost: "https://your-instance.thoughtspot.cloud",
  authType: AuthType.TrustedAuthToken,
  customCssUrl: "https://your-domain.com/css/liveboard-starter.css"
});

Or use one of the theme variations:

  • css/themes/dark-theme.css - Modern dark theme
  • css/themes/light-minimal.css - Clean minimal theme
  • css/themes/corporate-blue.css - Professional corporate theme

Try Different Themes

To see how different themes look, replace the CSS file link in the <head> section:

<!-- Starter Theme (default) -->
<link rel="stylesheet" href="../css/liveboard-starter.css">

<!-- Dark Theme -->
<link rel="stylesheet" href="../css/themes/dark-theme.css">

<!-- Light Minimal Theme -->
<link rel="stylesheet" href="../css/themes/light-minimal.css">

<!-- Corporate Blue Theme -->
<link rel="stylesheet" href="../css/themes/corporate-blue.css">
